What AI Can Do Today

Text & Code — ChatGPT, Claude, Gemini, Codex. Conversational AI that writes, edits, summarizes, and codes. Free tiers available — paid unlocks longer context and better models. Some run in the terminal (Claude Code, Codex CLI) and can read/write files directly.
Images & Video — Midjourney, DALL-E, Runway, Firefly. Generate and edit visuals from text prompts. Quality is production-ready for many use cases.
Design & Prototyping — Figma AI, Galileo, v0. AI is entering the design tool workflow directly — generating layouts, components, and prototypes.
Code & Animation — AI-powered code editors (Cursor, Windsurf) and frameworks like Remotion let you generate programmatic animations and even render to video — all driven by code AI can write for you.
What they all share — They work best when you're specific. Vague input = generic output. The skill is learning to direct them.

Let's get hands-on with one specific thing AI is great at — turning static icons into animated ones.

Meet the SVG

  • Every SVG is just shapes described in coordinates
  • The viewBox is the coordinate system — animations use these numbers
  • Each shape can be animated independently

How it was built

  1. Exported the icon from Figma as SVG
  2. Identified the 4 shapes and their stroke lengths
  3. Asked Claude to animate them sequentially with draw-on
  4. Iterated: adjusted timing overlaps so shapes cascade smoothly

The prompt that built this:

"Read caglia.svg. It has an outer circle, top circle, bottom circle, and a horizontal oval. Animate each sequentially with stroke-dashoffset draw-on. Outer draws first (0-25%), top circle next (15-40%), bottom circle (30-55%), oval last (45-70%). Hold all visible 70-80%, then erase. Total cycle 4s, loop."

Pick Your Tool

Claude Code / Codex CLI Terminal-based. Reads files, writes SVGs, creates previews. Best for batch work and Figma MCP integration.
Claude.ai / ChatGPT Upload SVG in chat, describe what you want. Simpler, no setup — but no file system access.
Gemini / Google AI Studio Google's model. Strong at code generation, generous free tier. Can work with SVG markup directly in chat.
Remotion React-based programmatic video/animation. AI writes the code, Remotion renders to MP4. For when SVG/SMIL isn't enough.
HeroUI Studio New Visual AI canvas for SVG animation. Node-based workflow: upload SVG → write a text prompt → AI generates the animation. Built-in presets (fade, scale, spin, bounce), easing controls, and iterative chaining. No code required — closest to a designer-native tool.

Good Prompt vs Bad Prompt

Weak

"Animate this SVG"

Claude will guess. You'll iterate 3-4 times.

Strong

"Read spiral.svg. It has an outer circle and two inner circles centered at (15,15). Add a gentle pulse to the inner circles — animate r ±1 unit, 3s duration, stagger 0.3s. Keep outer circle static. Loop forever."

Specific elements, named technique, exact values, clear constraints.

Iteration Tips

  • "Draw-on is too fast — increase dur from 2s to 4s" (not "make it slower")
  • "Waves overflow clipPath — reduce translate from 5 to 3 units"
  • "Stagger feels jerky — reduce begin offset from 1s to 0.3s"
